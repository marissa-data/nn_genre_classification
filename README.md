# Using Neural Networks to Classify Songs into Musical Genres
Authors: Marissa Burton, Mohini Gupta, Sophie Kim, Shaveen Saadee

Repository contains two supervised neural network models for classification of musical genres. The first model is a Convolutional Neural Network (CNN) model trained on mel-spectrograms of audio waves (image classifcation task), while the second model is a Multi-Layer Perceptron (MLP) Neural Network model trained on raw .wav audio files transformed into Mel-Frequency Cepstral Coefficients (MFCCs; audio classifcation task).
