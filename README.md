Neural Network Modeling for Musical Genre Classification
Authors: Marissa Burton, Mohini Gupta, Sophie Kim, Shaveen Saadee

Repository contains two supervised neural network models for classification of musical genres. The first model is a Convolutional Neural Network (CNN) model trained on mel-spectrograms of audio waves (image classification task), while the second model is a Multi-Layer Perceptron (MLP) Neural Network model trained on raw .wav audio files transformed into Mel-Frequency Cepstral Coefficients (MFCCs; audio classification task).
